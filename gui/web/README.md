# PortMap-AI Web Dashboard Foundation

This directory contains a lightweight local HTML dashboard foundation for future browser-based operator views.

It is not a replacement for the existing Textual TUI. It does not include a frontend build system, network client, write endpoints, cloud sync, automatic enforcement, router changes, or active background probing.

Use `gui.web.sample_data.sample_dashboard_api_data()` for sanitized local previews.
